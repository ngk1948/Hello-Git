Ngôn ngữ lập trình yêu thích:
C++ là ngôn ngữ lập trình em yêu thích vì có hiệu năng cao, tính linh hoạt và được sử dụng rộng rãi trong thực tế. C++ cho phép lập trình cả theo hướng thủ tục và hướng đối tượng, giúp người học rèn tư duy logic và thuật toán rất tốt.
